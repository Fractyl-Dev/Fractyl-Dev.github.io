Morse Version 2.0: New Features and UI Redesigns

By Hunter Ball
January 17th, 2018

Hello everyone. Recently, FractylDev released Morse Version 2.0, which added some needed features. However, some of the users started experiencing critical bugs and issues in the app, crashing many people's apps. Our development team has been working hard to fix these issues, and today, Morse version 2.1 was released.

This new update fixes a few bugs that caused SMS messages to no longer be recieved, as well as some other small bugs. It also fixes the UI in the learn page so nothing is cut off.

Morse is available on the Google Play store for Android phones running Android 6.0 or later. Make sure to get it!
