What is Morse?

In a time where the complexity of technologies is ever-increasing, much merit can still be found in the adaptation of old methods to new circumstances.


MORSE does just this.


By adapting the language used on the now obsolete telegraph, Morse can provide users unprecedented functionality.


With this pioneering application, users can learn Morse Code, send text messages, and receive text messages (via vibration feedback). From technicians needing a hands-free communication option or handicapped persons seeking a way to use SMS messaging, MORSE has the potential to impact countless users.



Functions and Features

Communicating with Contacts:

-Viewing SMS messages in the SMS inbox.

-Texting contacts and automatically translating messages into Morse Code.


Learning Morse Code:

-Listening to Morse Code via vibration feedback and audio with a full Morse Code alphabet.

-Playing word games to practice identification of single characters, four-letter words, or five-letter words in Morse Code.


Adjusting settings:

-Customizing receipt of messages to indicate the full message, just the contact name, or only the contact first initial.

-Modifying Morse Vibration Speed to match skill level.
